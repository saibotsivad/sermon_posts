=== Sermon Posts ===
Contributors: saibotsivad
Tags: church, sermon, sermons, preaching
Requires at least: 3.0
Tested up to: 3.3.2

A plugin for WordPress to manage your sermons with ease! View and edit MP3 files and MP3 tags, including the elusive albumart. Uses built-in WordPress functionality, so styling is a snap!

== Description ==

Sermon Posts was originally conceived as a drop-in replacement for an outdated sermon management plugin, but after development it was clear that additional tools were desired.
